# 配置前端环境
---

### 检查node版本
``` 
npm -v
10.9.2
```
### 安装Vite
```
npm create vite@latest my-vue-app
```
此时vite会提示你选择
1️⃣Select a framework 👉 选择 Vue
2️⃣ Select a variant 👉 选择 TypeScript

然后安装pnpm
```
npm install -g pnpm
```
检查是否安装成功
```
pnpm -v
```

进入项目(项目我已经建好了,上传到项目里了)
```
cd Front/unilife
```
